# Overview

Documentation for blank panel PCB designs for modular synthesizers.

## Project Features

### Simple PCB Design

- **No electronic components** - Just blank PCB panels
- **Mounting holes** - Standard Eurorack mounting
- **Custom silkscreen** - Optional artwork and labeling
- **Edge.Cuts** - Precise panel dimensions

### Manufacturing

- **JLCPCB** - Designed for easy ordering
- **Standard PCB** - 1.6mm FR-4 material
- **Various HP widths** - Multiple panel sizes available
